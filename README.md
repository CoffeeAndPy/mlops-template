Este repositorio es será una plantilla de cookiecutter para proyectos de Python. Proporciona una estructura básica y configuraciones iniciales para facilitar el inicio de nuevos proyectos.
## Estructura del Proyecto
La estructura del proyecto es la siguiente:

```
.
├── .gitignore
├── .venv
├── README.md
├── requirements.txt
├── cookiecutter.json
└── {{ cookiecutter.project_slug }}
    ├── src
    │   └── __init__.py
    ├── tests
    │   ├── __init__.py
```
